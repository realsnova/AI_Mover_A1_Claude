# 單字冒險 使用說明書

> 適用對象：家長、老師與小玩家（國小三～六年級）。
> 最後更新：2026-07-12（Phase 6 商業級強化完成版）

---

## 一、這是什麼？

「單字冒險」是一個 **Cambridge English A1 Movers 英語單字學習遊戲**：

- **398 個單字**，對照劍橋官方 2025 年 A1 Movers 詞表**字面 100% 覆蓋**，每字都有中文、例句（中英對照）、圖示。
- 用「答對英文單字 → 收服寶可夢」的方式練習，共 **10 個主題區域**（動物、食物、身體、學校、運動、家庭、天氣、交通、時間數字、綜合挑戰）。
- 內建**間隔重複學習系統（SRS）**：答對的字會依 1→2→4→7→15 天的科學間隔安排複習，答錯的字打回重學——這是背單字 App（如 Anki）採用的同一套記憶原理。
- 免帳號、免安裝、免費；進度自動存在瀏覽器裡。

## 二、快速開始

### 方式 1：直接開網址（最簡單）

用 Chrome 或 Edge 開啟：

```
https://realsnova.github.io/AI_Mover_A1_Claude/
```

會自動進入遊戲。第一次載入需要網路。

### 方式 2：安裝成 App（建議手機/平板使用）

1. 用手機瀏覽器開啟上面的網址。
2. **Android（Chrome）**：點選單 → 「加入主畫面」或「安裝應用程式」。
   **iPhone/iPad（Safari）**：點分享按鈕 → 「加入主畫面」。
3. 之後從桌面圖示 🎮 開啟，就是全螢幕的 App 體驗。

### 方式 3：離線遊玩

**只要曾經上線開啟過一次**，遊戲會自動把需要的資源存進裝置快取，之後**沒有網路也能完整遊玩**（含怪獸圖與圖示）。若某隻怪獸的圖從未載入過，離線時會以 👾 圖示代替，不影響作答。

> 有新版本上線時，畫面下方會出現「🔄 有新版本可用」橫幅，點「立即更新」即可，不會弄丟進度。

## 三、遊戲玩法（給小玩家）

### 基本流程

1. **🗺️ 地圖**：從「新手草原」開始。每個區域先點「📖 先學習單字」把單字看過一輪（有發音）。
2. **⚔️ 戰鬥**：野生寶可夢出現！答對 5 題就能收服牠。題目有四種：
   - **看圖選字**：看圖選出正確英文
   - **聽音選圖**：聽發音選出正確圖片（要開聲音喔）
   - **拼字**：點字母方塊拼出單字（第一個字母已放好）
   - **句子填空**：把句子填完整
3. **🔴 收服**：答完 5 題就收服成功，寶可夢加入 📕 圖鑑。你越熟練牠身上綁定的單字，牠還會**進化**！
4. **🏟️ 道館**：收服該區 5～6 隻後可挑戰道館：10 題、每題限時 20 秒、答對 8 題過關拿**徽章**，解鎖下一個區域。
5. **👑 結局**：集滿 10 枚徽章成為單字冠軍，之後開啟自由挑戰模式。

金幣拿去 **🛒 商店** 買帽子和戰鬥背景裝扮自己！

### 特別玩法

- **📋 每日任務**（地圖上方）：每天「學 3 個新字＋複習 5 個字」，完成領獎勵。每天做一點點，最有效！
- **✨ 閃光寶可夢**：連續登入 3 天以上，當天第一場戰鬥會遇到閃光色的稀有寶可夢！
- **📖 錯題本**（進度頁裡）：常答錯的字會集中在這裡，點「開始複習戰」專門練它們。
- **🎤 跟我唸**（學習模式裡）：對著麥克風唸出單字，遊戲會聽你唸得對不對（需要 Chrome/Edge 並允許麥克風）。

## 四、家長與老師使用建議

### 建議的學習節奏

**每天 10～15 分鐘、完成每日任務即可，貴在天天玩。**

遊戲的間隔複習系統（1/2/4/7/15 天）就是為「每天少量」設計的——今天答對的字，系統會自動在最容易遺忘的時間點再考一次。每天 15 分鐘的效果，遠勝過週末一次玩兩小時。

### 各年級的使用定位

| 年級 | 建議 |
|------|------|
| 小三～小四 | 主線玩法照序推進即可，一區一區解鎖不用催進度 |
| 小五 | 可鼓勵挑戰道館限時模式、衝「總精熟度」 |
| 小六（程度好） | A1 對其偏易，適合快速刷完當總複習；再進階需等 A2 Flyers 擴充（見升級路線） |

### 家長怎麼看學習成效

1. **📊 進度頁（儀表板）**：一眼看到學過字數、總精熟度、連續登入天數、各主題的精熟度條。
2. **📖 錯題本**：孩子的弱點字都在這，建議每週陪孩子清一次。
3. **📋 匯出學習紀錄**：進度頁點「匯出學習紀錄」，會把每個字的複習階段、答對答錯次數複製成表格（可貼到 Excel），適合想細看的家長或老師。

### 教室使用小技巧

- **設定面板（⚙️）**可分別開關「背景音樂／音效／單字發音」——教室投影時建議**關背景音樂、留單字發音**。
- 道館的 10 題限時模式很適合投影出來全班搶答。
- 一台裝置＝一份進度。多位學生共用同一台裝置時，請善用「存檔碼」輪流保存各自進度（見下一節）。

## 五、存檔與跨裝置

- **自動存檔**：進度隨時自動存在該瀏覽器的 localStorage，關掉重開都在。
- **手動存檔碼（重要）**：標題畫面或頂欄的「💾 存檔」可以產生一串**存檔碼**，複製保存起來。在任何裝置點「📂 讀檔」→ 貼上代碼即可還原進度。
  - 換手機、換電腦、清了瀏覽器資料、多人共用裝置——都靠它。
  - **建議每週備份一次存檔碼**（貼到記事本或傳給自己）。
- **重置進度**：地圖最下方的「🧹 重置進度」會清空一切重新開始（有二次確認，不會誤觸）。

## 六、疑難排解

| 狀況 | 解法 |
|------|------|
| 沒有聲音 | ① 檢查 ⚙️ 設定裡三個開關 ② 裝置音量 ③ 單字發音需要系統有英文語音（絕大多數手機/電腦都有） |
| 沒有「聽音選圖」題型 | 這是刻意設計：裝置沒有英文語音、或你關閉了「單字發音」時，聽力題會自動停用，避免聽不到聲音卡關 |
| 怪獸圖變成 👾 | 該圖尚未快取且目前離線；連網再玩一次該區即可 |
| 沒看到「🎤 跟我唸」按鈕 | 該瀏覽器不支援語音辨識（例如 Firefox、部分 iPhone），屬正常，不影響其他功能 |
| 進度不見了 | 多半是換了瀏覽器/裝置或清了瀏覽器資料——用之前備份的存檔碼還原 |
| 畫面下方出現「有新版本」 | 點「立即更新」即可，進度不受影響 |
| 第一次載入很慢 | 首次需下載怪獸圖與圖示；之後有快取就快了，離線也能玩 |

## 八、技術與版權摘要

- **架構**：單一 HTML 主程式＋PWA（manifest + Service Worker），部署於 GitHub Pages。離線策略：自家資源「網路優先、離線退快取」，怪獸圖/圖示「快取優先」。
- **品質驗證**：五種螢幕寬度（375–1920px）版面實測、真實點擊完整戰鬥流程自動化測試、離線重載測試、Lighthouse 稽核（正式站 Best Practices 100 分）。
- **隱私**：`robots.txt` + `noindex` 雙重防搜尋引擎索引；無任何追蹤、不蒐集任何資料。
